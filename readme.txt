Trivial Pursuit type card game.


Loads all questions and answers data into a queue
which is called the deck from "trivial.txt". User answers questions from
the deck or discard pile. If answered correctly question
is placed at bottom of deck (queue). Incorrect answers go on the 
top of the discard pile (stack). Players can choose to answer from
the discard pile if it exists. When score of 5 points is 
reached the game is over.


Please enjoy and change as much as you want.
